TReX OS

Cretaceous Edition — A Unity-powered Linux distribution focused on elegance, minimalism, performance, and full user control.

Project Status

How It Started

TReX OS began as a learning challenge. Built by a Mechanical Engineer driven by curiosity, the project started from a simple question: “Can I build my own Linux distribution from scratch?”

With the assistance of AI as a structured development partner, this project demonstrates that disciplined learning and persistence can overcome traditional barriers to entry in software development.

Vision & Position

TReX OS is built on Ubuntu 24.04 LTS, leveraging its stability, security updates, and ecosystem strength.

We respect the Unity desktop revival and broader Linux community. TReX OS does not compete — it refines.

The long-term vision is a clean, independent distribution with a strong identity and disciplined design philosophy.
