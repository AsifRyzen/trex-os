# TReX OS

TReX OS is a community-driven Unity-focused Linux distribution based on Ubuntu LTS.

## Mission

To provide a clean, stable, modern Unity desktop experience while supporting the broader Unity ecosystem.

TReX OS does not aim to compete with Ubuntu Unity, but to collaborate and contribute through testing, polishing, and structured experimentation.

## Philosophy

- Ubuntu LTS based
- Unity only
- Stability first
- Snap daemon disabled by default
- Community oriented
- Non-commercial

## Build Instructions

1. Install Ubuntu 24.04 Server
2. Clone this repository
3. Run:
./build-trex.sh
